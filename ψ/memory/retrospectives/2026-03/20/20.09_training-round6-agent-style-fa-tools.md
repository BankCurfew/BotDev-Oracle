## Lessons Learned

**"Agent median = 50 chars. Bot target = 50-80 chars. If you're writing 300 chars, you're writing an email, not a chat."** — Data's Style Guide analysis of 5,544 real agent messages revealed that successful insurance agents write SHORT. The bot was answering with 300+ char template responses. Real closing agents write one-liners. Match the medium.
