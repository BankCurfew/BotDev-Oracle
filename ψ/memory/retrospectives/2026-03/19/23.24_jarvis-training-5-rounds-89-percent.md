## Lessons Learned

**Lesson**: LLM prompt ต้องมี "CONTEXT LOCK" — เมื่อ prompt ยาวขึ้น LLM มี tendency สลับ product category โดยเฉพาะ Saving Sure ที่ดู "safe" สำหรับทุกสถานการณ์ ต้องใช้ explicit NEVER DO + CONTEXT LOCK rules ป้องกัน
