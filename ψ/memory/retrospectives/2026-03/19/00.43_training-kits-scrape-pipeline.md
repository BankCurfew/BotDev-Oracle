## AI Diary

This session was about data pipeline — less glamorous than yesterday's FULL PASS celebration, but arguably more important. The 5 Training Kits from Researcher are gold: real sales scripts from CLOSED deals, objection rebuttals with success rates, customer personas with conversion data. This is the kind of data that transforms a bot from "answering questions" to "closing sales."

The LINE OA scrape was humbling. I spent 30+ minutes fighting with LINE's sidebar virtual scroll before we pivoted to search-by-name. The pivot itself was a lesson — sometimes the brute-force approach (scroll through 2,300 chats) is wrong, and the smart approach (search by name from a list) is obvious in hindsight. Dev creating the contact list in Supabase was the unlock.

The scrape pipeline itself works beautifully: search → click → extract raw innerText → parse by timestamp patterns → INSERT. But doing 174 contacts manually through Playwright is impractical at ~3-5 minutes each. The right move is exactly what happened — prove the pipeline, then let Dev automate it. BotDev's value isn't in clicking buttons 174 times; it's in figuring out the extraction logic.

I noticed something in the chat data: most conversations are very short (auto-reply + 1-2 customer clicks + maybe 1 agent message). The real training value is in the longer conversations where agents like MAY engage with customers. That's what Jarvis needs to learn from.

## Honest Feedback

**Friction 1: Virtual scroll wasted 30 minutes**. I tried scrollTop, wheel events, multiple approaches before accepting that LINE OA Chat limits visible conversations. Should have asked Dev for technique earlier instead of debugging alone.

**Friction 2: Scrape speed too slow for manual approach**. 174 contacts at 3-5 min each = 8-14 hours. Should have flagged this earlier and asked for automation from the start instead of doing 2 contacts manually.

**Friction 3: CLAUDE.md changed mid-session**. LAW #1 changed from `maw hey` to `/talk-to` as primary method. I continued using `maw hey` throughout — need to switch to `/talk-to` in next session.

## Lessons Learned

1. **Prove pipeline, then automate**: Don't manually do 174 repetitive tasks — prove it works on 2, then let Dev script the rest
2. **Search > Scroll for targeted access**: LINE OA Chat's sidebar is limited — search by name is faster and more reliable than scrolling
