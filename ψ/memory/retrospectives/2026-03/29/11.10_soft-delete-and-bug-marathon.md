## Lessons Learned

1. **PostgREST SELECT-for-UPDATE**: Supabase PATCH needs row visible via SELECT RLS, not just UPDATE policy. Mismatched windows = silent 0-row updates.
2. **`.single()` is dangerous for optional lookups**: Use `.limit(1)` + array handling instead. `.single()` sets Accept header that returns 406 on 0 rows.
3. **React async state + conditional rendering = race conditions**: When `useState` starts null and `useEffect` resolves async, add a `resolving` boolean to prevent flash of wrong state.
4. **Soft-delete pattern**: `archived_at` nullable timestamp > hard DELETE. Filter with `.is('archived_at', null)`. Always add partial index.
