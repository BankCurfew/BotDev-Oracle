## Lessons Learned

1. **Coordination > Code**: BotDev's highest leverage is orchestrating 8+ oracles, not writing code alone
2. **Test before overwriting**: FA Tools key swap broke auth — always test new credentials before replacing old ones
3. **Check .env before announcing ports**: Port 3200 not 3100 — verify infrastructure before telling the team
4. **FULL PASS requires systematic QA**: 93 benchmarks, 4 critical issues, 3 prompt improvements = quality
