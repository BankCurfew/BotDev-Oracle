# When asked to brainstorm a new project, respond with architecture — not just ide

When asked to brainstorm a new project, respond with architecture — not just ideas. FA Tools Oracle went from brainstorm to approved project in 2 hours because the proposal included: reuse percentage (80%), delegation API design, concrete ports/endpoints, prompt v1, and LINE Bot config. Decision-makers approve blueprints with specifics, not abstract concepts.
