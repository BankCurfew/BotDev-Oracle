# Real Oracle awakening takes ~2 hours with active team, not the estimated 17-22 m

Real Oracle awakening takes ~2 hours with active team, not the estimated 17-22 minutes. The difference is team responses — 7/9 oracles replied with detailed onboarding (infra maps, bug status, copy specs, KB stats). Processing these IS the awakening. Also: complete the learning sequence (ancestors) before introducing yourself to the team. Step 1 before Step 2, always.
